## 1. 서비스 개요

### 서비스명
**K-Culture Explorer** (케이컬처 익스플로러)

### 한 줄 소개
> 한국의 문화유산을 발견하고, 탐방 경험을 나누고, 나만의 여행 루트를 AI가 설계해주는 플랫폼

### 핵심 가치
| 가치 | 설명 |
|------|------|
| 🏛️ **발견 (Discover)** | 한국 문화유산의 매력을 쉽고 깊이 있게 소개 |
| 📝 **공유 (Share)** | 실제 탐방 후기와 사진으로 생생한 경험 공유 |
| 🗺️ **안내 (Guide)** | AI 기반 맞춤 루트 추천 및 준비 가이드 |

### 서비스 구조 (3-Pillar)

```
┌─────────────────────────────────────────────────────┐
│                  K-Culture Explorer                  │
├─────────────────┬──────────────────┬────────────────┤
│  🏛️ K-Culture   │  📝 탐방 후기     │  🗺️ 루트 플래너 │
│    소개 섹션     │    커뮤니티       │   (AI 기반)     │
├─────────────────┼──────────────────┼────────────────┤
│ • 문화유산 DB    │ • 후기 작성/조회  │ • 목적지 선택   │
│ • 카테고리 탐색  │ • 사진 갤러리    │ • 최적 루트 생성 │
│ • 역사/배경 설명 │ • 평점/좋아요    │ • 준비물 안내   │
│ • 영상/이미지   │ • 댓글/소통      │ • 예산/시간 안내 │
└─────────────────┴──────────────────┴────────────────┘
```

## 3. 기능 명세서 (MVP)

### P0 - 핵심 기능 (Must Have)

| 기능 | 설명 | 사용자 스토리 |
|------|------|--------------|
| **문화유산 소개 페이지** | 카테고리별(궁궐/사찰/성곽/마을 등) 문화유산 소개 | 사용자로서 한국 문화유산을 카테고리별로 탐색하고 싶다 |
| **상세 정보 뷰** | 역사, 사진, 위치, 운영시간, 입장료, 접근성 정보 | 사용자로서 방문 전에 상세 정보를 한눈에 보고 싶다 |
| **탐방 후기 작성** | 방문 후기 + 사진 업로드 + 별점 평가 | 사용자로서 내 탐방 경험을 다른 사람과 공유하고 싶다 |
| **후기 조회/검색** | 유적지별 후기 목록, 인기순/최신순 정렬 | 사용자로서 다른 사람의 후기를 참고하고 싶다 |
| **AI 루트 플래너** | 목적지 선택 → AI가 최적 동선/준비물/예산 안내 | 사용자로서 가고 싶은 곳을 선택하면 루트를 자동으로 받고 싶다 |

### P1 - 중요 기능 (Should Have)

| 기능 | 설명 |
|------|------|
| **지도 뷰** | 문화유산 위치를 지도에서 시각적으로 탐색 |
| **좋아요/북마크** | 관심 유적지 저장 |
| **다국어 지원** | 영어/일본어/중국어 기본 지원 |
| **계절별 추천** | 현재 시즌에 맞는 추천 유적지 |
| **교통 안내** | 대중교통/자가용 접근 방법 상세 안내 |

### P2 - 추가 기능 (Nice to Have)

| 기능 | 설명 |
|------|------|
| **AR 체험** | 유적지 현장에서 AR로 역사 해설 |
| **스탬프 투어** | 방문 인증 스탬프 수집 게이미피케이션 |
| **소셜 공유** | SNS 자동 공유 카드 생성 |
| **오디오 가이드** | AI 음성 해설 제공 |
| **코스 공유** | 내가 만든 루트를 다른 사용자와 공유 |

## 4. AI 루트 플래너 상세 설계

### 입력 (사용자가 선택)
```
1. 🏛️ 목적지 (1~5개 선택)
2. 📅 여행 날짜 / 기간
3. 🚗 이동 수단 (대중교통/자가용)
4. 👥 인원 (혼자/커플/가족/단체)
5. 💰 예산 범위 (선택)
```

### 출력 (AI가 생성)
```
📍 최적 방문 순서 & 동선
⏰ 각 장소별 추천 소요 시간
🚌 이동 방법 & 예상 소요 시간
🎒 계절/날씨에 맞는 준비물 리스트
💰 예상 비용 내역 (입장료, 교통비, 식비)
🍽️ 주변 맛집/카페 추천
📸 포토 스팟 추천
⚠️ 주의사항 & 팁
```

## 5. 시장 분석

### TAM / SAM / SOM

| 구분 | 규모 | 근거 |
|------|------|------|
| **TAM** (전체 시장) | 약 25조원 | 한국 국내 관광 시장 (2025 기준) |
| **SAM** (접근 가능 시장) | 약 3조원 | 문화유산/역사 관광 세그먼트 |
| **SOM** (목표 시장) | 약 300억원 | 디지털 문화관광 플랫폼 사용자 (1년 내) |

### SWOT 분석

| | 긍정적 | 부정적 |
|---|--------|--------|
| **내부** | **강점**: AI 기반 맞춤 루트, 커뮤니티 후기, 다국어 지원 | **약점**: 초기 콘텐츠/후기 부족, 브랜드 인지도 없음 |
| **외부** | **기회**: K-Culture 해외 열풍, 정부 관광 디지털화 정책 | **위협**: 네이버/카카오맵 기존 서비스, 관광공사 플랫폼 |

### 경쟁사 분석

| 서비스 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|-------------|
| 대한민국 구석구석 (관광공사) | 공식 데이터, 신뢰성 | UI 올드, 개인화 없음 | AI 맞춤 루트 |
| 네이버 지도 | 대규모 리뷰, 길찾기 | 문화유산 특화 X | 문화유산 전문 큐레이션 |
| 트리플 | 여행 일정 관리 | 문화유산 깊이 부족 | 역사/문화 심층 콘텐츠 |
| Visit Korea | 외국인 대상 | 한국인에 최적화 X | 내/외국인 통합 서비스 |

### 시장 진입 전략
1. **1단계**: 서울 5대 궁궐 + 주요 유네스코 세계유산 집중
2. **2단계**: 전국 주요 문화유산 확대 + 커뮤니티 활성화
3. **3단계**: 다국어 확장 + B2B (여행사/교육기관) 파트너십

## 7. 기술 아키텍처 제안

```
┌── Frontend ──────────────────────────────────────┐
│  Next.js / React + 지도(Kakao/Naver Map API)      │
├── Backend ───────────────────────────────────────┤
│  FastAPI + LangChain (AI 루트 플래너)              │
│  PostgreSQL (유적지 DB, 후기 DB)                   │
│  S3/CloudFlare R2 (이미지 저장)                    │

│  GPT / Gemini / Claude (루트 생성, 콘텐츠 생성)    │
│  Embedding + Vector DB (유사 유적지 추천)          │
└──────────────────────────────────────────────────┘
```
