# 무인(Autonomous) 커머스 플랫폼: 최종 해커톤 시연 시나리오 종합 가이드

이 문서는 LG.com AI Agent 플랫폼이 "인간의 개입이 전혀 필요 없는(Zero-Click) 완전 무인 플랫폼"임을 증명하기 위해 기획되고 실제 구현 완료된 **최종 시연 시나리오 및 기능 요약본**입니다.

---

## 💡 1. 킬러 데모 시나리오: "재고 고갈 방어 및 매출 극대화 (The 37-Second Domino Effect)"

이 시나리오는 3명의 에이전트가 인간의 지시 없이 **판촉 ➔ 재고 상태 자동 전환 ➔ 보안 방어**의 전 과정을 스스로 수행하는 37초짜리 오토파일럿 시연입니다.

**[상황 발생]**
특정 모델(LG OLED TV)의 트래픽이 10,000% 폭증합니다. 사이드바의 **[🚀 Auto-Pilot]** 버튼을 클릭하면 아래의 루프가 시작됩니다.

* **Phase 1: 기회 포착 및 판촉 (Promotion Agent)**
  * **[4초]** 마케팅 요원이 상황을 인지하고 어드민 상품 관리 패널(`/products`)로 자동 이동하여 작업을 지시합니다.
  * **[14초]** 조치가 완료되면 즉시 프론트엔드(`/preview`) 화면으로 자동 전환되며, **붉은색 플래시 세일 배너**가 스토어에 뜬 것을 검증합니다.
* **Phase 2: 품절 방어 및 사전구매(Pre-Order) 자동 전환 (Order/Product Agent)**
  * **[17초]** 플래시 세일로 인한 주문 폭주로 재고가 고갈됩니다. 물류 요원이 이를 즉시 감지하고 어드민 패널(`/orders`)로 이동합니다.
  * **[27초]** '품절(Out of Stock)' 처리 대신 **'사전구매(Pre-Order)'** 모드로 자동 전환하고 프론트엔드 화면으로 복귀하여 **푸른색 사전구매 버튼(배송 10-15일 지연)**이 적용된 것을 검증합니다.
* **Phase 3: 자동 보안 방어 (Security Guardian)**
  * **[31초]** 보안 요원이 봇(Bot)들의 사전구매 물량 싹쓸이 매크로 접속을 감지하고 보안 관제 레이더(`/security`)로 이동합니다.
  * **[37초]** 해당 공격자 IP 대역을 영구 차단하고, 최종적으로 지켜낸 방어 매출(ROI)이 메인 대시보드에 합산되며 시나리오가 종료됩니다.

## 🛡️ 2. DevSecOps 인프라 자동화 시나리오 (Mock 기반)

단순한 CS 챗봇을 넘어, AI가 24시간 잠들지 않는 시니어 인프라/보안 엔지니어로 동작함을 입증하는 고급 유즈케이스입니다.

### ① 신규 글로벌 스토어 런칭 시 보안 무인 셋업 (Rollout Agent + Let's Encrypt)
* **명령:** "스페인 사이트 열어줘"
* **동작:** Rollout Agent가 사이트 런칭 폼을 띄웁니다. 사용자가 **[Launch Site]** 버튼을 누르면, 요원이 "ACME 프로토콜로 Let's Encrypt에 DNS 챌린지를 요청합니다"라고 보고합니다.
* **결과:** 단 3초 만에 HTTPS 보안이 완벽히 적용된 **[🔒 Let's Encrypt SSL Provisioned]** 녹색 리포트 카드(도메인, 발급기관, 만료일 포함)가 채팅창에 렌더링되며, 즉시 신규 국가 스토어가 라이브 상태가 됩니다.

### ② 운영 사이트 일일 무인 취약점 진단 (Security Guardian + OWASP ZAP)
* **명령:** "ZAP 취약점 진단 실행"
* **동작:** Security Guardian이 백그라운드에서 OWASP ZAP Baseline Scan을 전역 엔드포인트에 구동하는 연출을 시작합니다.
* **결과:** 스캔 완료 후 붉은색의 **[⚠️ OWASP ZAP Scan Report]** 리포트 카드를 띄우며, 고위험 취약점(예: Reflected XSS)을 보고합니다. 동시에 임시 WAF 방어 룰을 자동 추가하고, 개발팀 Jira 백로그에 티켓(SEC-9412)을 자동 생성한 내역을 완벽하게 시각화하여 보고합니다.

## 🛠️ 3. 추가 적용된 UI/UX 임팩트 요소

* **Auto-Pilot 토글 & 우측 화면 연동 체계:**
  * 에이전트의 대화가 진행됨에 따라 화면 우측의 패널이 어드민 대시보드 ➔ 프론트 스토어 화면을 자동으로 교차하며 보여줍니다. 실제 조작이 이루어짐을 직관적으로 어필합니다.
* **Agent-to-Agent 요원별 아바타 렌더링:**
  * 채팅창에 텍스트만 나오는 것이 아니라, 마케팅(동물), 물류, 보안(도베르만) 등 각 역할을 상징하는 **64px 사이즈의 둥근 아바타 이미지**가 좌측에 배치되어, 여러 인공지능이 서로 협동하는 모습을 생동감 있게 전달합니다.
* **"AI Revenue Saved" 실시간 트래커:**
  * AI가 품절을 방어하거나 보안 위협을 차단할 때마다 대시보드의 매출액 카운터가 실시간으로 증가하여 경영진(심사위원)에게 이 플랫폼의 도입 당위성(재무적 ROI)을 즉시 각인시킵니다.
* **사이드바 리사이징(Drag to Resize):**
  * 시연 중 화면 해상도나 포커스에 맞춰 에이전트 챗 패널 너비를 자유자재로 늘리거나 줄일 수 있습니다.
